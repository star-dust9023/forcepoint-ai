# Forcepoint Enterprise AI skills catalogue

## What skills are
Skills are reusable, governed packages that give the Enterprise AI assistant
proven ways to do specific work — for example querying Salesforce efficiently or
triaging a DLP incident. Skills are authored, validated, and published through a
governed pipeline and stored in the internal GitHub skills repository.

## Salesforce CData Query Cache (GA)
Eliminates redundant discovery calls when querying Salesforce, delivering roughly
85 percent token reduction per typical query. Available to all authorised
enterprise users.

## DLP Incident Response Triage (in pilot)
Assists DLP analysts in classifying and prioritising incidents using behavioural
context and policy history.

## Pipeline Intelligence Assistant (in design)
Surfaces deal risk signals and suggests next-best actions for sales reps using
CRM and activity data.

## Policy Q&A — HR & Legal (in design)
Answers employee questions about Forcepoint policies using approved knowledge
sources, with a zero-hallucination approach via grounded retrieval.

## Contributing a skill
Anyone can propose a skill through the portal's skill intake. Submissions are
routed into the GitHub repository for triage and validation before publication.
